# Future Scope: Kids Creativity & Growth Features

**Date:** 2026-04-12
**Status:** Planned (not started)

## Overview

Features to enhance kids' growth, humor, and creativity on the platform. To be built incrementally — each feature gets its own spec and implementation cycle.

## Recommended Build Order

### 1. Drawing Canvas + Avatar & Badge System (Foundation)
- **Drawing Canvas**: Built-in HTML5 canvas where kids can draw and submit artwork for challenges. Brush sizes, colors, eraser, undo. Save as image, attach to submissions.
- **Avatar & Badge System ("আমার চরিত্র")**: Kids earn badges — "Story Writer", "Joke Master", "Quiz Champion", "7-Day Streak". Customizable avatar that levels up with activity. Tied to actual creative output, not gamification for its own sake.
- **Why first**: Canvas is a tool everything else uses. Badges are the reward backbone.

### 2. Daily Creative Challenges ("আজকের চ্যালেঞ্জ")
- Daily prompt: "Draw a fish that lives in the sky", "Write a 3-line funny story about a cat"
- Kids submit via canvas (drawing) or text
- Gallery wall where parents can view/share
- Earns badges for streaks and participation
- **Depends on**: Canvas, Badge System

### 3. Joke of the Day ("হাসির আসর")
- Daily Bengali joke/riddle curated by team
- Kids can submit their own jokes (moderated)
- Voting: "Was this funny?" — teaches humor appreciation
- Weekly "Funniest Joke" badge
- **Depends on**: Badge System
- **Effort**: Small

### 4. Creative Portfolio ("আমার সৃষ্টি")
- Each child gets a portfolio page: drawings, stories, poems
- Parents can share the portfolio link (mini website for the kid)
- Aggregates work from challenges, canvas submissions, stories
- Builds pride in work, encourages more creation
- **Depends on**: Challenges, Canvas

### 5. Interactive Story Builder ("গল্প বানাও")
- Choose-your-own-adventure style stories in Bengali
- Branching paths with choices
- Kids can create their own branching stories for others to play
- Teaches cause & effect, imagination, writing
- **Depends on**: Badge System
- **Effort**: Large (save for last)

## Additional Ideas
- **DIY Activity Cards ("নিজে বানাও")**: Printable craft/experiment cards with video companions. Sells as physical card packs in shop.
- **Bengali Word Games ("শব্দ খেলা")**: Daily word puzzles, rhyme completion, tongue twisters with audio.
- **Debate/Opinion Corner ("তুমি কী মনে করো?")**: Weekly fun topics, kids pick sides and write arguments.
